+ (void)pingKinveyWithBlock:(KCSPingBlock)completionAction

Parameters

completionAction

The callback to perform on completion.

Discussion

Ping Kinvey and perform a callback when complete.

This method makes a request on Kinvey and uses the callback to indicate the completion, if you wish to check to see if the Kinvey Service is alive and responding, please use checkKinveyServiceStatusWithAction:

Warning: The results passed to completionAction have changed, to get the old style, initialzie Kinvey with the KCS_USE_OLD_PING_STYLE_KEY (or, if you’re using a plist, “kcsPingStyle”) key set to YES in your options.
